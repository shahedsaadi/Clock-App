# Overview:

 A website that displays the current time dynamically, the page would feature a digital clock showing the current time, which updates every second.

 # Technology Stack:
- Node.js
- JavaScript.
- React.js.
- useState React Hook.

# To Run The Project:
- Clone the repository (git clone https://github.com/shahedsaadi/Clock-App.git).
- Open the folder(repository) inside Visual Studio or any source-code editor.
- Open Terminal > cd your-destination-folder.
 
1) npm install
  
2) npm run dev

3) Visit the link: http://localhost:5173/
